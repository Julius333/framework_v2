Mijn Framework

Met een 'accordeon' effect kan je content uitklappen

Je kan gemakkelijk content verdelen in verschillende paragrafen om overzichtelijkheid te behouden.

Hoe zit het in elkaar?

Dit effect is gemaakt door een aantal inklapbare en uitklapbare items. Maar één item word tegelijk getoond.

Je kan elk type content plaatsen binnen dit framework. Bijvoorbeeld een UL-lijst:

  • Item 1
  • Item 2
  • Item 3

Hoe gebruik je het?

Je gebruikt het framework door simpelweg de HTML, CSS en Javascript te kopiëren. In de HTML staat waar je je content in kan vullen en hoe je alles uit kan breiden.

Je kan zoveel secties plaatsen als je wilt!
